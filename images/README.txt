Drop project screenshots (and your hero photo) in this folder.

For the big hero photo next to your name, add:

  hero.jpg   (roughly 4:3, e.g. a nice landscape/lifestyle photo of you)

For project cards, name each file to match the "image" path in script.js:

  bootleggers.png
  cigar-maps.png
  discord-soccer.png
  betrayal.png
  bestof5.png
  gibbs-prudential.png
  imad-website.png
  jaidyn-website.png

Tips:
- Use the same aspect ratio for all (16:9 works best with the layout).
- Compress them (aim under ~300 KB each) so the page loads fast.
- .png, .jpg, or .webp all work — just update the extension in script.js.
- Until a matching image exists, the card shows a "Screenshot coming soon" placeholder.
